# Serviço Perto — marketplace de serviços locais

Planejamento e análise multidisciplinar para um **marketplace mobile-first** que conecta **contratantes** a **prestadores** com **geolocalização**, perfil com mídia e categorias, **assinatura mensal** para o prestador aparecer na plataforma e mecanismos de **confiança** (avaliações, depoimentos). O escopo discute descoberta estilo **cartões/proximidade** e/ou **catálogo com busca e filtros**, além de opções de **pagamento após execução** do serviço.
